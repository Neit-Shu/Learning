# 3D Maze Ball

Учебный проект на Unity: управление шариком в лабиринте.

## Описание
Простой 3D-лабиринт с системой сбора предметов. Проект создан для изучения основ Unity:
- Работа с физикой (Rigidbody)
- Навигация по сценам
- Основы UI
